# Akan Name Generator

## Description
The Akan Name Generator is a simple web application that calculates the day of the week a user was born and assigns them a corresponding Akan name based on Ghanaian culture. Akan names are traditionally given based on the day of the week a person is born and their gender.

## Author
Warda Haithar

## Setup Instructions
1. Clone or download the repository to your local machine  
2. Open the project folder in Visual Studio Code  
3. Open the `index.html` file in a browser using Live Server or directly in the browser  
4. Enter your birth date and select your gender to get your Akan name  

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Accepts user birth date input (day, month, year)
- Allows user to select gender
- Calculates day of the week from input
- Displays the correct Akan name based on gender and birth day
- Validates user input for correctness

## BDD (Behavior Driven Development)

### Input:
- Day of birth
- Month of birth
- Year of birth
- Gender

### Process:
- Validate user input
- Calculate the day of the week using a mathematical formula
- Match the result to an Akan name based on gender

### Output:
- Displays the day of the week
- Displays the corresponding Akan name
